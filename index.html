Política de Privacidad

Última actualización: 25 de octubre de 2023

1. Recopilación de Información

La aplicación "Código Procesal Civil PE" utiliza servicios de terceros (Google AdMob) que pueden recopilar información para mostrar anuncios, como:

  • ID de publicidad (Advertising ID).
  • Datos del dispositivo (modelo, sistema operativo).
  • Dirección IP.

2. Uso de los Datos

Los datos se utilizan para:

  • Mostrar anuncios personalizados.
  • Medir el rendimiento de la aplicación.

3. Terceros

Utilizamos Google AdMob para publicidad. Consulta su política de privacidad: Política de Privacidad de Google.

4. Seguridad

No almacenamos datos personales sensibles de los usuarios.

5. Derechos del Usuario

Puedes desactivar la personalización de anuncios en la configuración de tu dispositivo.

6. Contacto

Para consultas, contáctanos en: tucorreo@jobalistudios.com.
